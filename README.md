# 主要内容：
- 這個網頁是全客戶端（也就是説，沒有任何數據會傳到伺服器）
- 你只需要上傳一個 64x32 或者 64x64 的玩家皮膚照片，就可以開始修改咯~
- 目前有想過要不要弄多國語言版，但想想還是算了（反正也沒多少人用）
